# ApexSandboxAndroidTV

Android TV (Leanback) sandbox for exercising CTV SDK flows. Emulator only.

Requirements
- Android Studio Iguana+ (AGP 8.5+)
- JDK 17

Quick start
1. Open this folder in Android Studio.
2. Create an Android TV emulator (Android TV 1080p, API 34).
3. Run the "tvapp" configuration on the emulator.
