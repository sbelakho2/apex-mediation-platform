# Support

## Contact Us

If you encounter any issues or have questions about integrating Apex Mediation, please reach out to our support team.

*   **Email**: [support@apexmediation.com](mailto:support@apexmediation.com)
*   **Developer Portal**: [https://developer.apexmediation.com](https://developer.apexmediation.com)

## Reporting Issues

When reporting a technical issue, please include the following information to help us resolve it faster:

1.  **App ID** and **Placement ID**.
2.  **SDK Version** (e.g., Android 1.0.0).
3.  **Device Model** and **OS Version**.
4.  **Steps to Reproduce** the issue.
5.  **Logs**: Attach verbose logs from the device (see [Debugging Guide](11_GUIDE_DEBUGGING.md)).
